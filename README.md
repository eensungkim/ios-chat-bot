## 구현 세부사항

- Core Graphics 를 활용한 말풍선 그리기  
- Core Animations 을 활용한 로딩 인디케이터  
- Modern Collection Views 를 활용한 채팅뷰 구현  
  - Diffable Data Source 와 Snapshot 의 활용  
- MVVM 적용
